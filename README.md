# 📊 Inteligencia de Negocios – Caso Muebles Aconcagua

Este proyecto simula una solución de Business Intelligence para la empresa ficticia **Muebles Aconcagua**, desarrollada como parte de una formación en análisis de datos e inteligencia de negocios.

---

## 🧱 Estructura del Proyecto

🔹 Modelo dimensional tipo estrella  
🔹 Carga ETL automatizada con **SQL Server Integration Services (SSIS)**  
🔹 Dashboard interactivo desarrollado con **Power BI**  

---

## 🗃 Herramientas Utilizadas

- SQL Server 2019
- SQL Server Management Studio (SSMS)
- SSIS (Integration Services)
- Power BI Desktop

---

## 🧠 Preguntas de Negocio

- ¿Qué productos fueron más comprados por solteros en Viña del Mar en 2015?
- ¿Cuáles fueron los productos más vendidos por sucursal?
- ¿Cuál fue el ranking de precios promedio por categoría?
- ¿Cuántas ventas generó cada vendedor en enero de 2015?

---

## 📈 Resultados

El dashboard desarrollado permite obtener insights relevantes para la toma de decisiones, como:
- Identificar productos más vendidos por región
- Medir el desempeño comercial por categoría y vendedor
- Detectar patrones de compra según perfil de cliente

---

## 📌 ¿Cómo usar este proyecto?

1. Restaurar el archivo `.bak` en SQL Server.
2. Ejecutar el ETL para cargar datos al Data Warehouse.
3. Abrir el archivo `.pbix` con Power BI Desktop y conectar la fuente si es necesario.
